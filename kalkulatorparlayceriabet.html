Panduan Perhitungan Single Bet
Pahami cara menghitung setiap skenario taruhan
Full Menang (Win)
Semua odds dikalikan dengan Nominal Bet. Modal kembali + keuntungan.
Profit = Odds x Nominal Bet - Nominal Bet
Total = Odds x Nominal Bet
Contoh: Odds 1.85, Nominal Bet 100.000
Profit = 1.85 x 100.000 - 100.000 = Rp 85.000
Total diterima = Rp 185.000
Menang Setengah (Win Half)
Profit dihitung setengah dari odds normal. Modal kembali + profit setengah.
Profit = (Odds x Nominal Bet - Nominal Bet) / 2
Total = Nominal Bet + Profit
Contoh: Odds 1.85, Nominal Bet 100.000
Profit = (1.85 x 100.000 - 100.000) / 2 = Rp 42.500
Total diterima = 100.000 + 42.500 = Rp 142.500
Seri (Draw/Void)
Tidak ada profit maupun kerugian. Seluruh modal taruhan dikembalikan ke akun.
Profit = 0
Total = Nominal Bet (kembali penuh)
Contoh: Nominal Bet 100.000
Total diterima = Rp 100.000 (modal kembali)
Kalah Setengah (Lose Half)
Kehilangan setengah dari modal taruhan. Setengah modal dikembalikan.
Kerugian = Nominal Bet x 0.5
Total = Nominal Bet x 0.5 (kembali setengah)
Contoh: Nominal Bet 100.000
Kerugian = 100.000 x 0.5 = Rp 50.000
Total diterima = Rp 50.000
Kalah (Lose)
Kehilangan seluruh modal taruhan. Tidak ada yang dikembalikan ke akun.
Profit = 0
Total = 0 (hilang semua)
Contoh: Nominal Bet 100.000
Total diterima = Rp 0
Odds Minus (American)
Odds minus artinya taruhan lebih besar dari keuntungan. Profit = Nominal Bet x (100 / |Odds|)
Profit = Nominal Bet x (100 / |Odds|)
Total = Nominal Bet + Profit
Contoh: Odds -150, Nominal Bet 150.000
Profit = 150.000 x (100 / 150) = Rp 100.000
Total = 150.000 + 100.000 = Rp 250.000

Panduan Perhitungan Parlay
Rumus lengkap untuk setiap skenario parlay
Rumus Dasar Parlay
Profit = (Total Perkalian Odds - 1) x Nominal Bet
Total Diterima = Profit + Nominal Bet
Semua Menang (W)
Semua odds dikalikan, dikurangi 1, lalu dikalikan Nominal Bet.
Odds Efektif = Odds1 x Odds2 x ... x OddsN
Profit = (Odds Efektif - 1) x Nominal Bet
Contoh: 3 partai odds 1.80, 2.00, 1.90, Nominal Bet 50.000
Odds Efektif = 1.80 x 2.00 x 1.90 = 6.84
Profit = (6.84 - 1) x 50.000 = Rp 292.000
Ada Win Half (WH)
Odds partai WH diubah: (Odds - 1) / 2 + 1
Odds WH = (Odds - 1) / 2 + 1
Contoh: 3 partai: W(1.80), WH(1.90), W(2.00), Nominal Bet 50.000
Odds WH = (1.90 - 1) / 2 + 1 = 1.45
Odds Efektif = 1.80 x 1.45 x 2.00 = 5.22
Profit = (5.22 - 1) x 50.000 = Rp 211.000
Ada Draw (D)
Odds partai Draw dianggap 1 (tidak mempengaruhi perhitungan).
Odds D = 1 (diabaikan dalam perkalian)
Contoh: 3 partai: W(1.80), D(2.00), W(1.90)
Odds Efektif = 1.80 x 1 x 1.90 = 3.42
Profit = (3.42 - 1) x 50.000 = Rp 121.000
Ada Lose Half (LH)
Partai LH mengurangi setengah stake dari parlay. Odds tetap normal, tapi multiplier jadi 0.5.
Multiplier LH = 0.5
Odds Efektif = Odds lain x 0.5
Contoh: 3 partai: W(1.80), LH(2.00), W(1.90)
Odds Efektif = 1.80 x 0.5 x 1.90 = 1.71
Profit = (1.71 - 1) x 50.000 = Rp 35.500
Note: Setengah stake kalah, setengah lanjut.
Ada Kalah (L)
Jika ada 1 saja partai yang kalah (L), seluruh parlay dianggap kalah.
Profit = 0
Total = 0 (hilang semua)
Contoh: 3 partai: W(1.80), L(2.00), W(1.90)
Hasil = KALAH TOTAL
Total diterima = Rp 0

Penjelasan Format Odds
Pahami perbedaan setiap format odds
Decimal (Eropa)
Format paling umum. Total payout = Odds x Nominal Bet.

Contoh: Odds 1.85, Nominal Bet 100.000
Total = 1.85 x 100.000 = Rp 185.000
Profit = 185.000 - 100.000 = Rp 85.000
American (US)
Positif (+): Keuntungan per 100 unit Nominal Bet.
Negatif (-): Nominal Bet yang dibutuhkan untuk menang 100 unit.

Contoh (+185): Nominal Bet 100.000 -> Keuntungan 185.000
Contoh (-120): Nominal Bet 120.000 -> Keuntungan 100.000
Fractional (UK)
Ditulis sebagai pecahan (contoh: 17/20).
Keuntungan = (Numerator/Denominator) x Nominal Bet

Contoh: 17/20, Nominal Bet 100.000
Keuntungan = (17/20) x 100.000 = Rp 85.000
Hong Kong
Sama dengan Decimal dikurangi 1.
Total = (HK Odds + 1) x Nominal Bet

Contoh: HK Odds 0.85, Nominal Bet 100.000
Total = (0.85 + 1) x 100.000 = Rp 185.000
Malay
Range Malay: -1.00 sampai +1.00.

Positif: Keuntungan per 1 unit Nominal Bet.
Negatif: Nominal Bet per 1 unit keuntungan.

Rumus dari HK:
Jika HK > 1 -> Malay = 1 / HK
Jika HK <= 1 -> Malay = HK

Contoh (0.85): Nominal Bet 100.000 -> Keuntungan 85.000
Contoh (-1.18): Nominal Bet 118.000 -> Keuntungan 100.000
Contoh HK 23.99: Malay = 1 / 23.99 = 0.0417
Indo
Sama dengan Hong Kong, format yang umum di Indonesia.

Contoh (0.85): Nominal Bet 100.000 -> Keuntungan 85.000
Contoh (-1.18): Nominal Bet 118.000 -> Keuntungan 100.000
